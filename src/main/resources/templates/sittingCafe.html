👩‍💻 역할 및 Tech Stacks

  • 서비스 기획 및 총괄
  • 아두이노를 이용해 실시간 인원 수 카운팅과 혼잡도 계산 및 웹과의 연동 구현
  • 백엔드(PHP, MySQL, JSON 사용) 개발

🙋‍♀️ 구현 내용

  • Arduino
    • Wemos D1R1 Board

      WIFI 모듈이 내장되어있어 무선통신이 가능

    • 적외선 센서

      본체에 2개의 센서를 연결하여 각각 들어오고 나가는 사람 counting

    • GET Request를 통하여 Server에 실시간으로 Data 전송

      Data

      • 센서로부터 측정되는 두 개의 데이터
      • 카페 고유의 ID와 각 카페별 최신 Data값 구별을 위한 uid

  • PHP
    • 아두이노로 부터 GET 방식을 통해 값을 받은 후, DB Table에 삽입
    • 실시간 인원 정보는 또다른 DB 테이블은 사용하지 않고 PHP 내에서 처리
    • 인원수 측정에 있어 오차를 극복하기 위해 비율을 정하여 혼잡도로 표시
  • JSON

    Android로 값을 넘겨줄 땐 JSON 형식으로 넘겨줘야하기 때문에 간단한 처리를 해주었음
